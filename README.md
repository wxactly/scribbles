# [Scribbles](http://wxactly.com/scribbles)

Generative art scribbled in javascript.

http://wxactly.com/scribbles

## Getting Started

```sh
$ npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt watch
```
